Git Commands

  1. cd (folder)- for changing directories
  2. cd .. - for reverting to prev directory
  3. touch (itemname) - create new file in directory
  4. git init - initializes local rep
  5. ls OR la - to check files in directory
  6. git status - self-explanatory
  7. git add (existing 'touched' filename)- adds it to local rep
  8. git commit -m (message)- pushes commits locally
  9. Now create a GitHub Rep of the same names and follow the instructions to connect the remote version with the local one
